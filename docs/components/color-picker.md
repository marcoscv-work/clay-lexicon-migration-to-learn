# Color Picker

Use the color picker to let users select a color from a predefined palette,
enter a hexadecimal value, or build a custom color. The field pairs a color
square (which opens the palette panel) with a text field for the hex value.

## Variants and anatomy

- **Predefined palette panel**: a set of colors you configure. Provide at least
  seven. It can also give access to custom color creation.
- **Custom color panel**: color cells, a color map, RGB inputs, a hue slider,
  and the hex field. All of them stay in sync as any one changes.

Variations combine the two panels: restricted (predefined only), unrestricted
(predefined plus up to twelve custom colors), and simple (hex field only).

Try the [Color Picker story](https://storybook.clayui.com/?path=/story/design-system-components-colorpicker--default),
the [custom colors story](https://storybook.clayui.com/?path=/story/design-system-components-colorpicker--custom-colors),
and the [native story](https://storybook.clayui.com/?path=/story/design-system-components-colorpicker--native).

## Usage guidelines

- Restrict the palette when brand or accessibility constraints apply, and let
  users define custom colors only when they truly need arbitrary values.
- Keep the hex field editable; many users arrive with an exact value.
- Show the current selection in the field's color square at all times.

## Implementation

Install the component and the CSS package:

```shell
npm install @clayui/css @clayui/color-picker
```

Control the value; pass `colors` for the predefined palette and enable custom
colors with `showPalette` and `onColorsChange`:

```jsx
import ColorPicker from '@clayui/color-picker';
import {useState} from 'react';

export default function Example() {
	const [color, setColor] = useState('0B5FFF');

	return (
		<ColorPicker
			colors={['0B5FFF', '287D3C', 'B95000', 'DA1414']}
			label="Accent color"
			onValueChange={setColor}
			showHex
			value={color}
		/>
	);
}
```

## Accessibility

- The trigger and every palette cell are keyboard operable, and each color cell
  carries an accessible name (the color's name or value), not only the swatch.
- The hex input gives a color-independent way to set the value.
- If the chosen color encodes meaning elsewhere, ensure that meaning does not
  rely on color alone.
