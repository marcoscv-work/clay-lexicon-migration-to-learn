# Layout Components

These components implement the [grid](/foundations/grid) and
[layout](/foundations/layout) foundations in React: `Container` (fixed and
fluid), `Row`, `Col`, `Content` helpers for autofit rows, and `Sheet`, the form
box container.

## Variants and anatomy

- **Container and ContainerFluid**: the fixed and fluid containers from the
  grid foundation.
- **Row and Col**: the twelve column system, with sizing per breakpoint and
  ordering.
- **Sheet**: the form box (8 columns with offsets) used by
  [forms](/components/forms).

Try the [row positioning story](https://storybook.clayui.com/?path=/story/design-system-components-layout--row-positioning),
the [breakpoints story](https://storybook.clayui.com/?path=/story/design-system-components-layout--breakpoints),
and the [ordering story](https://storybook.clayui.com/?path=/story/design-system-components-layout--ordering).

## Usage guidelines

- Follow the layout foundation to choose the composition (cards, form box,
  full box, sidebar and box); these components express it in markup.
- Use the column system for medium and large blocks, not for spacing small
  elements (that is the 8px grid's job).

## Implementation

Install the component and the CSS package:

```shell
npm install @clayui/css @clayui/layout
```

```jsx
import ClayLayout from '@clayui/layout';

export default function Example() {
	return (
		<ClayLayout.ContainerFluid view>
			<ClayLayout.Row>
				<ClayLayout.Col md={8}>Main content</ClayLayout.Col>
				<ClayLayout.Col md={4}>Secondary content</ClayLayout.Col>
			</ClayLayout.Row>
		</ClayLayout.ContainerFluid>
	);
}
```

## Accessibility

- Layout components are structural and carry no semantics; keep landmarks and
  headings on the content they wrap.
- Column ordering changes visual order only; keep the DOM order matching the
  reading order.
