# Management Toolbar

Use the management toolbar on top of a dataset view (table, list, or cards) to
search, filter, sort, switch the view type, select items, and expose bulk
actions. It is a specialized [toolbar](/components/toolbar) and the companion of
every dataset in Liferay's applications.

## Variants and anatomy

The default composition places, left to right: the selection checkbox, the
filter and order drop down, the search field, and the view type and add
buttons. When items are selected, the bar switches to its **active state**,
showing the selection count and the bulk actions.

A **results bar** below the toolbar shows the applied filters as removable
labels and the "x results for" text with a clear action.

Try the [Management Toolbar story](https://storybook.clayui.com/?path=/story/design-system-components-managementtoolbar--default).

## Usage guidelines

- Pair the toolbar's selection with checkboxes in the dataset rows or cards;
  selection and actions always reflect in the bar.
- Follow the [Common Actions](/patterns/common-actions) ordering for the filter
  and order drop down and the actions menu.
- Show applied filters in the results bar so they can be removed individually
  or cleared at once.

## Implementation

Install the component and the CSS package:

```shell
npm install @clayui/css @clayui/management-toolbar
```

Compose the bar from its item list, search, and actions:

```jsx
import ManagementToolbar from '@clayui/management-toolbar';
import Button from '@clayui/button';

export default function Example() {
	return (
		<ManagementToolbar>
			<ManagementToolbar.ItemList>
				<ManagementToolbar.Item>
					<Button className="nav-link" displayType="unstyled">
						Filter and Order
					</Button>
				</ManagementToolbar.Item>
			</ManagementToolbar.ItemList>
			<ManagementToolbar.Search onSubmit={(event) => event.preventDefault()} />
			<ManagementToolbar.ItemList>
				<ManagementToolbar.Item>
					<Button displayType="primary">Add</Button>
				</ManagementToolbar.Item>
			</ManagementToolbar.ItemList>
		</ManagementToolbar>
	);
}
```

## Accessibility

- The search field is a labeled form control inside the bar.
- The active (selection) state announces the selection count.
- Filter labels in the results bar are removable with the keyboard, and each
  remove control names the filter it clears.
