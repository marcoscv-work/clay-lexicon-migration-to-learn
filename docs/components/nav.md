# Nav

Use Nav for basic navigation lists: the low level building block behind
navigation patterns such as pills, underline tabs, and stacked side
navigations. Higher level components build on it; reach for Nav directly when
none of them fits.

<!-- REVIEW: Clay-only building block; the Lexicon navigation specs are covered by Navigation Bar and Vertical Nav. -->

## Variants and anatomy

Compose `Nav` with `Nav.Item` and `Nav.Link`; the active link carries the
selected state. The nav can render horizontally or stacked vertically, and a
decorator variant highlights the active item for side navigation.

Try the [Nav story](https://storybook.clayui.com/?path=/story/design-system-components-nav--default)
and the [with decorator story](https://storybook.clayui.com/?path=/story/design-system-components-nav--with-decorator).

## Usage guidelines

- Prefer the specialized components first:
  [Navigation Bar](/components/navigation-bar) for application sections,
  [Vertical Nav](/components/vertical-nav) for form navigation,
  [Tabs](/components/tabs) for switching views.
- Keep exactly one active item.

## Implementation

The component ships in `@clayui/core` (re-exported by `@clayui/nav`):

```shell
npm install @clayui/css @clayui/core
```

```jsx
import {Nav} from '@clayui/core';

export default function Example() {
	return (
		<Nav aria-label="Site pages">
			<Nav.Item>
				<Nav.Link active href="#pages">
					Pages
				</Nav.Link>
			</Nav.Item>
			<Nav.Item>
				<Nav.Link href="#fragments">Fragments</Nav.Link>
			</Nav.Item>
		</Nav>
	);
}
```

### Nav

| Prop | Type | Default | Required | Description |
| ---- | ---- | ------- | -------- | ----------- |
| `nestMargins` | `boolean` |  | No | Flag to indicate if `nav-nested-margins` class should be applied. Adds margin to indent each nested navigation. |
| `nested` | `boolean` |  | No | Flag to indicate if `nav-nested` class should be applied. Adds padding to indent each nested navigation. |
| `stacked` | `boolean` |  | No | Flag to indicate if `nav-stacked` class should be applied. |

## Accessibility

- Wrap the nav in a labeled navigation landmark (`aria-label`) when the page
  has more than one.
- Mark the current item with `aria-current="page"` through the active link.
