# Navigation Bar

Use a navigation bar (navbar) inside an application to navigate between its
sections. It is always horizontal, with one or more entry points ordered left to
right by importance.

## Variants and anatomy

- **Default**: light background with dark text, defined for sites.
- **Inverted**: dark background with light text, defined for administration
  controls, always placed right below the header.

When the entries exceed the bar's width, the overflowing items collapse into the
last tab and open through a drop down menu; once the user picks an entry outside
the drop down, it collapses again.

Try the [Navigation Bar story](https://storybook.clayui.com/?path=/story/design-system-components-navigationbar--navigation-bar).

## Usage guidelines

- Place the most important entry points first.
- Keep exactly one item marked as active.
- Use the inverted variant in admin interfaces for contrast with the content
  below the header.

## Implementation

Install the component and the CSS package:

```shell
npm install @clayui/css @clayui/navigation-bar
```

Compose the bar with `NavigationBar.Item` entries; `triggerLabel` names the
collapsed menu on small screens:

```jsx
import NavigationBar from '@clayui/navigation-bar';
import Link from '@clayui/link';

export default function Example() {
	return (
		<NavigationBar inverted triggerLabel="Pages">
			<NavigationBar.Item active>
				<Link className="nav-link" href="/pages">
					Pages
				</Link>
			</NavigationBar.Item>
			<NavigationBar.Item>
				<Link className="nav-link" href="/fragments">
					Fragments
				</Link>
			</NavigationBar.Item>
		</NavigationBar>
	);
}
```

## Accessibility

- The bar renders inside a navigation landmark; give it an accessible name when
  a page has more than one nav (for example `aria-label="Application"`).
- Mark the current section on the active item so it is announced (for example
  with `aria-current="page"` on the active link).
- The collapsed trigger on small screens is a real button, keyboard operable,
  labeled by `triggerLabel`.
