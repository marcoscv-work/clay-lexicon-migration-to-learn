# Panel

Use a panel to group related content in a container with a header, a body, and
an optional footer. Collapsible panels help organize long settings or
configuration screens into sections the user can expand as needed.

<!-- REVIEW: Lexicon documents form sections rather than a standalone panel spec; the collapsible-section rules here come from the forms guidance (first level sections may collapse, second level never). Confirm with the design team. -->

## Variants and anatomy

A panel combines `Panel.Header`, `Panel.Body`, and `Panel.Footer`, and can be
marked `collapsable` so the header toggles the body. Panels group into stacks
with `Panel.Group`, and the secondary display type provides the bordered sheet
style used in forms.

Following the forms guidance, only first level sections should collapse; never
nest collapsible panels inside collapsible panels.

Try the [Panel story](https://storybook.clayui.com/?path=/story/design-system-components-panel--default),
the [collapsable story](https://storybook.clayui.com/?path=/story/design-system-components-panel--collapsable),
and the [groups story](https://storybook.clayui.com/?path=/story/design-system-components-panel--groups).

## Usage guidelines

- Use panels to chunk long configuration content; keep each panel's title short
  and descriptive.
- Do not hide required fields inside collapsed panels; collapsed content should
  be optional or advanced.
- Keep the expanded or collapsed state consistent with the user's task (for
  example expand the section with errors).

## Implementation

Install the component and the CSS package:

```shell
npm install @clayui/css @clayui/panel
```

```jsx
import Panel from '@clayui/panel';

export default function Example() {
	return (
		<Panel collapsable displayTitle="Advanced options" displayType="secondary" showCollapseIcon>
			<Panel.Body>Optional configuration fields.</Panel.Body>
		</Panel>
	);
}
```

## Accessibility

- The collapsible header is a real button exposing `aria-expanded`, keyboard
  operable with `Enter` and `Space`.
- The panel title labels the section, so navigation by headings and regions
  works.
- Content revealed by expanding is inserted after the header in reading order.
