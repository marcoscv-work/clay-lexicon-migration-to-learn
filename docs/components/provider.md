# Provider

Wrap your application once in the Clay `Provider` to supply what every
component needs: the icon spritemap path, the theme container, and global
behaviors. It aggregates the icon and modal contexts, so you configure them in
one place.

## Variants and anatomy

- **Spritemap**: components that render icons resolve the spritemap from the
  Provider instead of a per-component `spritemap` property.
- **Theme**: the Provider marks the container that scopes Clay's CSS themes.
- **Reduced motion**: JavaScript driven transitions respect the Provider's
  reduced motion setting; pure CSS transitions use the
  `c-prefers-reduced-motion` class. Components such as
  [Empty State](/components/empty-state) also accept static image alternatives.

Try the [spritemap story](https://storybook.clayui.com/?path=/story/design-system-application-provider--spritemap)
and the [theme story](https://storybook.clayui.com/?path=/story/design-system-application-provider--theme).

## Usage guidelines

- Mount one Provider at the application root; nest another only to change the
  configuration for a subtree.
- Set `reducedMotion` to follow the user's system preference unless the user
  chose an in-app setting.

## Implementation

The component ships in `@clayui/core`:

```shell
npm install @clayui/css @clayui/core
```

```jsx
import {Provider} from '@clayui/core';
import '@clayui/css/lib/css/atlas.css';

export default function App({children}) {
	return (
		<Provider reducedMotion="user" spritemap="/icons.svg">
			{children}
		</Provider>
	);
}
```

### Provider

| Prop | Type | Default | Required | Description |
| ---- | ---- | ------- | -------- | ----------- |
| `children` | `React.ReactNode` |  | Yes | The content of the Provider. |

| `spritemap` | `string` |  | Yes | Path to the location of the spritemap resource. |
| `storageMaxSize` | `number` | `20` | No | Set the amount of items that can be cached, set to zero will be treated as infinite, be aware to set an ideal size to offer a positive experience for your user but not use a large amount of memory. |

## Accessibility

- The reduced motion setting is an accessibility feature: honor the operating
  system preference by default and never force animations on users who opted
  out.
