# Resize Handle

Use the resize handle on the edge of a resizable panel, such as the
[vertical bar](/components/vertical-bar)'s panel, so users can adjust its width
by dragging or with the keyboard.

<!-- REVIEW: Clay-only utility component (no Lexicon spec). -->

## Variants and anatomy

The handle attaches to the left or right edge of the panel it resizes, within
the minimum and maximum widths you configure.

Try the [resizing left story](https://storybook.clayui.com/?path=/story/design-system-components-resizehandle--resizing-left)
and the [resizing right story](https://storybook.clayui.com/?path=/story/design-system-components-resizehandle--resizing-right).

## Usage guidelines

- Constrain resizing with sensible minimum and maximum widths, so the panel
  never becomes unusable.
- Persist the user's chosen width when the panel reopens.
- Show the resize cursor on hover so the affordance is discoverable.

## Implementation

The component ships in `@clayui/core`. The vertical bar integrates it through
its `resize` property:

```shell
npm install @clayui/css @clayui/core
```

```jsx
import {VerticalBar} from '@clayui/core';

export default function Example() {
	return (
		<VerticalBar panelWidthMax={500} panelWidthMin={280} resize>
			{/* content and panels */}
		</VerticalBar>
	);
}
```

### ResizeHandle

| Prop | Type | Default | Required | Description |
| ---- | ---- | ------- | -------- | ----------- |
| `aria-controls` | `string` |  | Yes | ID of the element being resized. |

| `maxWidth` | `number` |  | Yes | The maximum width allowed. |
| `minWidth` | `number` |  | Yes | The minimum width allowed. |
| `onWidthChange` | `(width: number) => void` |  | Yes | Callback called every time the width changes (controlled). |
| `position` | `'left' \| 'right'` |  | Yes | The side of the screen where the resized element is located. |
| `width` | `number` | `320` | No | Property to set the current width (controlled). |

## Accessibility

- The handle is keyboard operable: arrow keys resize in steps, within the
  configured limits.
- It exposes the separator role with the current value, so the size is
  announced while resizing.
