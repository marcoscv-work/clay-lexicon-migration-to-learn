# Sticker

Use a sticker to visually identify elements of a dataset that belong to the same
group, such as a file type on a document card or a person's initials next to a
comment. Stickers identify; they do not communicate counts (use a
[badge](/components/badge)) or status (use a [label](/components/label)).

## Variants and anatomy

### Shapes

- **Square**: identifies files and folders. It can hold up to three characters
  or an icon, always in uppercase.
- **Round (circle)**: identifies people only. It can hold up to two characters
  or an image, always in uppercase.

Try the [shape story](https://storybook.clayui.com/?path=/story/design-system-components-sticker--shape)
and the [user image story](https://storybook.clayui.com/?path=/story/design-system-components-sticker--user-image).

### Sizes

- **24px**: small contexts where the element only needs visual identification.
- **32px**: image cards, so more of the image stays visible.
- **40px**: user identification in comments. Round only.
- **48px**: user identification in blog entries and articles. Round only.

Try the [size story](https://storybook.clayui.com/?path=/story/design-system-components-sticker--size).

### Position

On a card, place the sticker at the bottom left. A sticker can also sit on the
edge or outside corner of its container; try the
[positions story](https://storybook.clayui.com/?path=/story/design-system-components-sticker--positions).

## Usage guidelines

- Stickers can be any color; the default is the secondary gray `#6B6C7E`. Use
  consistent color and icon pairings for known file types, so users learn them.
- Use uppercase for the characters inside a sticker.
- Use the round shape exclusively for people.

## Implementation

Install the component and the CSS package:

```shell
npm install @clayui/css @clayui/sticker
```

Set the color with `displayType`, the size with `size`, and compose the content
with an icon, characters, or an image:

```jsx
import Sticker from '@clayui/sticker';
import Icon from '@clayui/icon';

export default function Example() {
	return (
		<>
			<Sticker displayType="danger">PDF</Sticker>
			<Sticker displayType="info">
				<Icon symbol="document" />
			</Sticker>
			<Sticker shape="circle" size="lg">
				<Sticker.Image alt="Portrait of the document owner" src="/images/user.jpg" />
			</Sticker>
		</>
	);
}
```

## Accessibility

- When a sticker carries meaning (a file type, a person), give its content an
  accessible name: `alt` text on `Sticker.Image`, or visible characters that
  read correctly.
- A sticker is decorative identification; do not make it the only place where
  essential information lives.
