# Time Picker

Use the time picker to let users enter a time value, editing hours and minutes
(and the meridiem in 12 hour locales) as separate segments. Like the
[date picker](/components/date-picker), it targets desktop browsers; prefer the
operating system's native control on mobile devices.

When a form needs date and time together, use the date picker's combined
variant instead of two separate fields.

## Variants and anatomy

The field splits the value into segments (hours, minutes, meridiem). Each
segment is editable by typing or with the arrow keys, and spinner arrows appear
on hover and focus.

Try the [Time Picker story](https://storybook.clayui.com/?path=/story/design-system-components-timepicker--default).

## Usage guidelines

- Match the format (12 or 24 hours) to the user's locale.
- Let users type the value directly; the segments keep the input valid.
- Validate ranges (for example opening hours) at the field, disabling invalid
  choices where possible.

## Implementation

Install the component and the CSS package:

```shell
npm install @clayui/css @clayui/time-picker
```

Control the value as an object with the segments:

```jsx
import TimePicker from '@clayui/time-picker';
import {useState} from 'react';

export default function Example() {
	const [value, setValue] = useState({hours: '--', minutes: '--'});

	return <TimePicker onChange={setValue} values={value} />;
}
```

Use `use12Hours` for locales with meridiem.

## Accessibility

- Each segment is focusable and adjustable with the arrow keys, following the
  spinbutton pattern.
- The whole field needs a label associating it with its purpose ("Start
  time").
- Keep typed input working; do not force the pointer spinners.
